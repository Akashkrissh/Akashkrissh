## Featured Project

### [CyberSentinel](https://github.com/Akashkrissh/CyberSentinel--Real-Time-Emergent-Threat-Detection-for-AI-Powered-Security-Platforms) — AI-Powered Threat Detection & SOC Automation

An end-to-end security operations platform that automates detection, triage, and investigation reporting using ML and LLMs.

**What it does:**
- Ingests high-throughput security telemetry from Prometheus APIs and TimescaleDB; applies continuous rule-based and ML-driven anomaly scoring to surface actionable alerts
- Deploys an autonomous threat hunting agent with multi-step reasoning over network flow and user behaviour logs — forms and tests threat hypotheses without analyst intervention
- Automates alert triage and escalation: classifies severity, cross-references known threat patterns, generates structured incident tickets with IOC lists, affected assets, and recommended playbook steps
- LLM-powered RAG pipeline (FAISS + HuggingFace) maps detected anomalies to MITRE ATT&CK TTPs, extracts malware indicators and phishing signatures from a live threat knowledge base
- Produces structured natural-language investigation reports per incident — evidence chains, root-cause findings, and remediation actions — ready for analyst review or audit

**Stack:** Python · FastAPI · Prometheus · TimescaleDB · FAISS · HuggingFace · Go · Docker
